# ShineMoon Design Studio Portfolio

Welcome to the ShineMoon Design Studio Portfolio! This repository contains the source code for a portfolio website designed to showcase the art of video editing and cinematography. Explore the website to discover courses, tutorials, and live classes aimed at elevating your video editing skills.

## Features

- **Responsive Design**: Optimized for both desktop and mobile devices.
- **Navigation**: Smooth navigation with a collapsible mobile menu.
- **Content Sections**: Structured sections for Home, Team, Courses, Portfolio, and Contact.
- **Social Media Links**: Integrated links to YouTube, Instagram, Twitter, and Facebook.
- **Interactive Elements**: Dynamic search functionality and interactive quick links.

## Demo

You can view a live demo of the portfolio [here](https://shinemoon.vercel.app).

## Installation

To get a local copy up and running, follow these simple steps:

1. **Clone the repository**:
    ```sh
    git clone https://github.com/khushalrathore/shinemoon.git
    ```
2. **Navigate to the project directory**:
    ```sh
    cd shinemoon
    ```

## Usage

Simply open `index.html` in your preferred web browser to view the portfolio locally.
